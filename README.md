# Cyberpunk Internship Portfolio

Modern single-page portfolio for **Muhammad Arham Anees**, built for internship applications with a dark cyberpunk aesthetic and a featured Time Table project case study.

## Highlights

- Dark mode palette with neon blue accents (`#00D4FF`) and deep slate background (`#0A192F`)
- Glassmorphism navigation, case-study cards, and project panels
- Time Table project case study with problem, contribution, stack, and outcome
- Smooth reveal animations and subtle hover interactions
- Terminal-style intro + code-snippet background for a tech-themed look
- Fully responsive layout for desktop and mobile

## Run Locally

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
npm run preview
```

## What’s inside

- `index.html` - Portfolio sections, featured Time Table case study, and contact links
- `styles.css` - Cyberpunk theme, glassmorphism, responsive layout, and mobile-friendly cards
- `main.js` - Reveal-on-scroll and safe hover effects
- `package.json` - Vite scripts and dependencies

## Best use

Share this portfolio with internship applications, especially when you want to highlight:
